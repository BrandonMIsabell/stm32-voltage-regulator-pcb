# stm32-voltage-regulator-pcb
Routed a custom 5V linear regulator PCB in KiCad to power an STM32 Nucleo board, utilizing an LM358 op-amp and IRFZ44N MOSFET for closed-loop feedback control  Fabricated physical hardware from custom Gerber/Drill files and designed a matching 3D-printed enclosure in Autodesk Fusion using exported STEP models
